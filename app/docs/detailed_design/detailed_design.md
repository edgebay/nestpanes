## 一、概述

### 1.1 模块简介

- 本文档覆盖 NestPanes 的根节点设计、界面与布局框架设计（`app/gui/` 目录下的核心类）
- 文件管理模块：[DDD-002 文件管理模块详细设计](file_management.md)

### 1.2 设计目标

- 窗格系统高度可扩展，新的窗格类型只需继承 PaneBase 并注册即可
- 布局管理器能灵活处理任意层级的拆分和嵌套
- 布局状态可序列化/反序列化，支持保存与恢复

### 2.2 类详细说明

#### 类：`AppNode`

| 属性     | 说明                                             |
| -------- | ------------------------------------------------ |
| **文件** | `app/app_node.cpp/h`                             |
| **职责** | 应用根节点，初始化所有子系统并协调它们之间的关系 |

**职责概述：**
- 初始化所有子系统（设置、主题、翻译、布局）
- 处理应用生命周期事件（关闭、焦点等）

---

#### 类：`LayoutManager`

| 属性     | 说明                                                          |
| -------- | ------------------------------------------------------------- |
| **文件** | `app/gui/layout_manager.cpp/h`                                |
| **职责** | 管理整体布局结构，包括侧边栏和主区域的拆分，以及布局保存/恢复 |

**职责概述：**
- 管理左侧边栏、右侧边栏和主内容区域的布局
- 处理标签页的拖拽拆分操作
- 将布局状态序列化到 ConfigFile 并恢复
- 协调侧边栏的显示/隐藏切换

---

#### 类：`PaneManager`

| 属性     | 说明                                               |
| -------- | -------------------------------------------------- |
| **文件** | `app/gui/pane_manager.cpp/h`                       |
| **职责** | 管理窗格的创建、关闭、查找，维护所有活动窗格的引用 |

**职责概述：**
- 将窗格类型注册到 PaneFactory
- 通过 PaneFactory 创建新窗格
- 处理窗格关闭逻辑（清理资源、更新标签页容器）
- 支持按类型/标识符查找已有窗格

---

#### 类：`PaneFactory`

| 属性     | 说明                                           |
| -------- | ---------------------------------------------- |
| **文件** | `app/gui/pane_factory.cpp/h`                   |
| **职责** | 窗格注册中心和工厂，通过类型标识符创建窗格实例 |

**职责概述：**
- 提供方法供各模块注册窗格类型
- 提供方法根据类型标识符创建窗格实例

---

#### 类：`PaneBase`

| 属性     | 说明                             |
| -------- | -------------------------------- |
| **文件** | `app/gui/pane_base.cpp/h`        |
| **职责** | 所有窗格的基类，定义窗格通用接口 |

**接口定义：**
- `get_type()` - 返回窗格类型标识符（用于工厂创建和序列化）
- `get_title()` - 返回标签页上显示的标题
- `get_icon()` - 返回标签页上显示的图标
- `get_config_data()` / `apply_config_data()` - 获取/应用窗格数据（供布局保存/恢复使用）

---

#### 类：`AppTabContainer`

| 属性     | 说明                                               |
| -------- | -------------------------------------------------- |
| **文件** | `app/gui/app_tab_container.cpp/h`                  |
| **职责** | 标签页容器，支持拖拽拆分、关闭按钮、新建标签按钮等 |

**功能：**
- 标签页的拖拽重排和跨容器拖拽
- 拖拽到容器边缘触发拆分操作（上/下/左/右）
- 标签页关闭按钮和新建标签页按钮
- 标签页右键菜单（关闭、关闭其他等）
- 快捷键支持：Ctrl+T 新建、Ctrl+W 关闭、Ctrl+Tab / Ctrl+Shift+Tab 切换

---

#### 类：`MultiSplitContainer`

| 属性     | 说明                                                      |
| -------- | --------------------------------------------------------- |
| **文件** | `app/gui/multi_split_container.cpp/h`                     |
| **职责** | 支持任意层级水平/垂直拆分的容器，管理多个 AppTabContainer |

**功能：**
- 支持水平和垂直两种拆分方向
- 拖拽调整拆分比例
- 嵌套组合实现任意复杂布局
- 当子容器所有标签关闭时自动合并/移除空拆分区域

---
